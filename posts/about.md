# 소개

프론트엔드 엔지니어 이찬희 입니다. 최근에는 리액트와 타입스크립트를 사용하여 즐겁게 개발하고 있습니다.
UX/UI에 관심이 있으며 현재 다노에서 웹 프론트엔드 엔지니어로 재직중입니다.

# 요약

- 리액트, 타입스크립트로 개발하는 것을 좋아합니다.
- 디자인 시스템에 관심이 있습니다.
- UX/UI는 단순히 예쁘게 만드는게 전부가 아니라는 것을 알고 있습니다.
- 테스트 코드 작성을 통해 품질을 유지하고자 노력합니다.
